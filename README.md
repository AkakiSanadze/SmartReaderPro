# Smart Reader Pro

Smart Reader Pro is a minimalist, gesture-based reading application optimized for mobile devices. It features a distraction-free interface, offline support (PWA), and adjustable settings for a comfortable reading experience.

## Features

- **Progressive Web App (PWA)**: Installable on iOS/Android and works offline.
- **Gesture Control**: Swipe left/right to navigate content with smart vertical-scroll detection.
- **Customizable**: Dark/Light/Sepia themes and adjustable font sizes.
- **Text-to-Speech (TTS)**: Listen to your articles with built-in voice support.

## hosting on GitHub Pages

1. Upload the files to a GitHub repository.
2. Go to **Settings** > **Pages**.
3. Select the `main` branch as the source.
4. Your PWA will be live at `https://your-username.github.io/repo-name/`.
